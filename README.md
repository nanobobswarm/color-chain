# Color Chain

A web-based puzzle game where you chain same-colored dots to score points before the timer runs out.

## How to Play

- Click a dot to select it, then click adjacent same-colored dots to extend the chain
- Chains of 3 or more dots clear from the board and add to your score
- Each cleared dot adds 2 seconds to the timer
- Consecutive chains build combo multipliers
- Game ends when the timer reaches 0

## Running

```bash
npm install
npm start
```
